## Contribution

The roadmap is built using [XMind](https://xmind.app/). The project file can be found at `maui-developer-roadmap.xmind`. To modify it, download and open XMind from [here](https://xmind.app/download/), click **Open File** and choose the `.xmind` file. It will open the roadmap for you. Update it, then export an updated PNG, update the image in the README, and create a PR (please minify exported PNGs with [Compressor.io](https://compressor.io/compress)).

- Open a pull request with improvements
- Discuss ideas in issues
- Spread the word
